# TanStack AI SaaS Starter

Free public starter for building fast web apps on **TanStack Start + Cloudflare Workers**.

## Stack

TanStack Start · React 19 · Tailwind v4 · shadcn/ui · Drizzle ORM · Better Auth · Cloudflare Workers · Vitest

## What this free repo includes

- TanStack Start app shell
- Landing page
- Auth flows
- i18n
- SEO basics
- Privacy and terms pages
- Monitoring hooks
- Security headers

## What this free repo does not include

- Billing
- Credits
- Stripe integration
- Pricing plans
- Paid AI workflows
- Newsletter monetization

## Local development

1. Copy `.dev.vars.example` to `.dev.vars`
2. Install dependencies
3. Run migrations if needed
4. Start the dev server

## Deploy

Use Wrangler or your preferred Cloudflare Workers deployment flow.
